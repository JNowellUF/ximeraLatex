\section{Introduction}

The goal of the Ximera Project is to allow authors to write online interactive content though \LaTeX\ documents.  The \code{ximera} class provides the structure for a \LaTeX\ document to be displayed in a dynamic online environment.

Each \code{.tex} document written in the \code{ximera} class is an \emph{activity}.  Generally, an activity will be composed of \emph{content} and \emph{questions}.  Content is material served to the student, while questions require a response from the student.  The basic structure of an activity is:

\begin{lstlisting}
\documentclass{ximera}
\title{Our Sample Activity}
\begin{document}
Let's multiply!
\begin{question}
	What is $3 \times 2$? \answer{6}
\end{question}
\end{document}
\end{lstlisting}
Note that every \code{ximera} document must have a \code{title}.

\section{Content}

To include an image, use the \code{image} environment.
\begin{lstlisting}
\begin{image}
	\includegraphics{myPrettyPicture.png}
\end{image}
\end{lstlisting}

The image environment also supports very basic TikZ images, provided they do not rely on information outside of the \code{tikzpicture} environment.
\begin{lstlisting}
\begin{image}
	\begin{tikzpicture}
		TikZ code goes here.
	\end{tikzpicture}
\end{image}
\end{lstlisting}

To include a YouTube video, use the \code{youtube} command.  For all other videos, use the \code{video} command.
\begin{lstlisting}
\youtube{some-youtube-url}
\video{some-video-url}
\end{lstlisting}


There is support for interactive javascript content via the \code{interactive} environment:
\begin{lstlisting}
\begin{interactive}[interactiveContent.js]
	Some static content.
\end{interactive}
\end{lstlisting}

\section{Questions and Answers}

A \code{question} environment collects at least one \code{answer} from the student.
\begin{lstlisting}
\begin{question}
	What is $3 \times 2$? \answer{6}
\end{question}
\end{lstlisting}
Online this displays the question ``What is $3 \times 2$?'' followed by an answer box.  The answer box will accept any answer that evaluates to 6, such as \fbox{$12 - 6$}.

A \code{prompt} environment helps to give an online answer structure.  For example,
\begin{lstlisting}
\begin{question}
	If Jack earned 12 dollars and spent 3 dollars, how many dollars does Jack have left?
	\begin{prompt}
		Jack has \answer{9} dollars left.
	\end{prompt}
\end{question}
\end{lstlisting}
The \code{prompt} environment is hidden in \code{handout} mode.


\subsection{Questions with Multiple Parts}

A \code{question} can contain multiple \code{answer} commands, which must all be correct for the Ximera server to consider the question complete.

\begin{lstlisting}
\begin{question}
	There are \answer{60} minutes in an hour, \answer{24} hours in a day, 
		and \answer{7} days in a week.
\end{question}
\end{lstlisting}

To display parts of a question in a list, use \code{enumerate}.
\begin{lstlisting}
\begin{question}
	\begin{enumerate}
		\item How many minutes in an hour? \answer{60}
		\item How many hours in a day? \answer{24}
		\item How many days in a week? \answer{7}
	\end{enumerate}
\end{question}
\end{lstlisting}

To force an online student to complete the parts sequentially, add the \code{hide} key to enumerate.  Initially only the first \code{item} will be shown.  A correct submission to all answers in an \code{item} reveals the next one.  The question is considered complete only when all answers have been correctly submitted.

\begin{lstlisting}
\begin{question}
	\begin{enumerate}[hide]
		\item How many minutes in an hour? \answer{60}
		\item How many hours in a day? \answer{24}
		\item How many days in a week? \answer{7}
	\end{enumerate}
\end{question}
\end{lstlisting}


\section{Hints}

A \code{hints} environment is a list of hints.
\begin{lstlisting}
\begin{question}
	What is $3 \times 2$? \answer{6}
	\begin{hints}
		\item Try multiplying.
		\item Try 6.
	\end{hints}
\end{question}
\end{lstlisting}
Online this adds a ``Show Hint'' button to the question, which reveals the list of hints one at a time.
